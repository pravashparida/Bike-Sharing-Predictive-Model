# Bike-sharing-case-study,This assignment is a programming assignment wherein you have to build a Multiple Linear Regression model for the prediction of demand for shared bikes.A US bike-sharing provider BoomBikes has recently suffered considerable dips in their revenues due to the ongoing Corona pandemic. The company is finding it very difficult to sustain in the current market scenario. So, it has decided to come up with a mindful business plan to be able to accelerate its revenue as soon as the ongoing lockdown comes to an end, and the economy restores to a healthy state.
